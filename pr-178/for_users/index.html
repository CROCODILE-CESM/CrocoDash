User Information#

Welcome! This section contains documentation for using CrocoDash to set up regional MOM6 cases. Whether you’re just getting started or diving into specific features, you’ll find helpful information here.

Getting Started#

New to CrocoDash? Start with these guides:

  1. Structure - Understand how CrocoDash is organized and what each component does

  2. Forcing Configuration - Learn how to configure forcing and boundary conditions

  3. Available Datasets - See what data sources are available

“What compsets are available?”#

Check Available Compsets for valid compset options.

“What forcing options do I need?”#

Read Forcing Configuration to understand what your compset requires.

“What datasets can I use?”#

See Datasets for available data sources (TPXO, GLORYS, GEBCO, etc.).
